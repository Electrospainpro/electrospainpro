# QUALITY_STANDARD.md

# Estándar de Calidad

Toda ficha de producto deberá contener:

- Imagen
- Descripción propia
- Especificaciones
- Pros
- Contras
- ESP Score
- Productos relacionados
- Enlaces de afiliación
- Marca
- Categoría
- Subcategoría

---

Toda comparativa deberá contener:

- Tabla comparativa
- Ventajas
- Inconvenientes
- Recomendación final
- ESP Score
- Alternativas

---

Toda guía deberá contener:

- Introducción
- Explicación técnica
- Consejos profesionales
- Productos recomendados
- Preguntas frecuentes

---

Nunca se copiará contenido de fabricantes.
