# Normas Editoriales ElectroSpainPro

## Objetivo

Publicar contenido técnico, independiente y útil para profesionales.

---

## Reglas

1. Nunca copiar contenido del fabricante.

2. Justificar todas las recomendaciones.

3. Explicar ventajas e inconvenientes.

4. Utilizar lenguaje profesional.

5. Mantener el contenido actualizado.

6. Todas las fichas tendrán ESP Score.

7. Todas las comparativas tendrán un ganador.

8. Siempre ofrecer alternativas.

9. Pensar primero en el profesional.

10. La confianza vale más que una venta.
