# Estrategia de Contenidos - ElectroSpainPro

Crear el portal de referencia para profesionales del sector eléctrico, fotovoltaico, telecomunicaciones e instrumentación mediante contenido técnico de alta calidad.

- Empresas instaladoras
- Ingenieros
- Técnicos de mantenimiento
- Autónomos
- Estudiantes de FP
- Empresas industriales

# Principios editoriales

1. Nunca copiar contenido de fabricantes.
2. Explicar ventajas e inconvenientes reales.
3. Justificar siempre las recomendaciones.
4. Utilizar lenguaje profesional y claro.
5. Mantener la información actualizada.
6. Priorizar la utilidad para el profesional.
7. Todas las fichas tendrán un ESP Score.
8. Siempre ofrecer alternativas.
9. Contenido optimizado para SEO.
10. Construir autoridad antes que vender.
